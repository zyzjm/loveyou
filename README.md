# 💍 求婚网页 - 增强版

这是一个浪漫的求婚网页，具有多主题系统和特殊动画效果。

## ✨ 新增功能

### 1. 多主题系统（5种主题）
每次打开网页会自动选择一个主题（同一天保持相同主题）：

- **经典夜空** 🌃 - 原始的深蓝夜空主题
- **粉色浪漫** 💕 - 温馨的粉色渐变，带有漂浮的心形
- **紫色梦幻** 💜 - 梦幻的紫色渐变，带有流星效果
- **蓝色星空** 💙 - 深邃的蓝色星空，带有极光效果
- **樱花主题** 🌸 - 柔和的樱花色，带有飘落的花瓣

### 2. 特殊动画效果
- 🌠 流星划过（紫色梦幻主题）
- ❤️ 漂浮的心形（粉色浪漫主题）
- 🌌 极光效果（蓝色星空主题）
- 🌸 飘落的花瓣（樱花主题）

### 3. 主题烟花
每个主题都有专属的烟花颜色配置，与主题完美协调。

### 4. 访问统计
- 使用 localStorage 记录访问次数
- 每天自动切换新主题
- 可在浏览器控制台查看统计信息

### 5. 特殊日期自动主题
在特定节日会自动切换对应主题：
- 💝 情人节（2月14日）- 粉色浪漫主题
- 💕 七夕节 - 樱花主题
- 🎄 圣诞节（12月24-25日）- 蓝色星空主题
- 🎆 元旦（1月1日）- 紫色梦幻主题

### 6. 隐藏彩蛋
- 🎨 按 T 键：快速打开主题选择器
- 💕 输入 "LOVE"：触发特殊心形雨效果
- 🌟 首次访问会显示欢迎提示

## 🎮 使用方法

1. 直接打开 `index.html` 即可
2. 每天打开会显示不同的主题样式
3. 清除浏览器 localStorage 可以重置主题选择

### 快速切换主题

有三种方式可以切换主题：

1. **点击右下角的 🎨 按钮** - 打开主题选择面板
2. **按键盘 T 键** - 快速打开/关闭主题选择器
3. **浏览器控制台命令**：

```javascript
// 查看可用主题
console.log(Object.keys(THEMES));

// 切换到指定主题（例如：romantic）
localStorage.setItem('loveTheme', 'romantic');
location.reload();
```

## 🛠️ 自定义主题

编辑 `themes.js` 文件中的 `THEMES` 对象即可自定义主题。

### 主题配置说明

```javascript
themeName: {
    name: '主题名称',
    background: '背景色或渐变',
    cityColor: '城市剪影颜色',
    moonColor: '月亮颜色',
    starColor: '星星颜色',
    buttonColor: '按钮颜色',
    textColor: '文字颜色',
    fireworkColors: ['烟花颜色数组'] 或 'random',
    specialEffect: 'none' | 'hearts' | 'shooting-stars' | 'aurora' | 'petals'
}
```
